Create Online Course Site With MEAN Stack - Introduction

Introduction

Now, I write this post as a diary about create online course site. I intend to make a simple, easy to use. This project was inspired by some site like Udemy, Khanacademy and other online course site. This source was build on MEAN stack technology contains Mongodb, Expressjs, Angularjs, Nodejs. It’s a modern stack and I heard about this so much. I have not much experienced about web programming and MEAN stack. Before, I used Angularjs to some project (PHP backend or No back end) and read numerous tutorials. Others technology I am newbie. Luckily, I found mean.js open source. This help me straightforward to build something instantly. Therefore, This blog to learn some thing new step by step to build what I like. I hope, some people will interested and help me correct what I are wrong (may be grammar on post :) ). I will very appriciate who help me make this more perfect by contribute in this project.

What will I build?

As said above, I want to build something like Udemy or Khanacademy … but much simpler. This source will deploy for Vietnamese project. I want this can help others to published their courses. This will very helpful for teachers can teach by this site. My simple idea is teachers can manage your resource like video, document or create quiz (or addition activity I will build later) in this site as playlist for one subject. So students just need follow this resource to gain knowledges. May be they follow with teachers who teach them on class, or whoever they like.

Brain storm:

This site will have courses. Each course will contains sections which will continuously contains videos, documents or activities.

* Homepage: List of courses (show info, description and link to course introduction)
* Course Introduction: More detail on course info, teachers and a button to start learning.
* Course Learning: This will show activities, items to learn like what Udemy or Khanacademy do.
* Course Creating: This is a form which teachers can input their resource to make a course live.
* Others but maybe later like discustion ...

Next

In next post, I will talk more detail about Mean.js and Data modeling of Mongodb. It’s a something new a have learnt.
